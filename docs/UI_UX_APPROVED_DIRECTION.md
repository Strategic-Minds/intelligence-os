# UI/UX Approved Direction

Approved visual direction: black background, satin black cards, white/light-gray typography, thin metallic silver borders, deep electric-blue hover/focus glow, green active status lights, red inactive/off status lights.

Approved image path in this package:

- `public/approved-ui/intelligence-os-approved-dashboard.png`
- `public/approved-ui/intelligence-os-approved-dashboard.svg`

Implementation rule: UI components must match this approved direction before adding feature complexity.

## Dashboard layout

- Left: black chat-only column, chat input fixed to bottom.
- Center: horizontal Kanban workflow board with workflow phases and rectangular step cards.
- Below center: Tools and Accounts sections.
- Right: black Google Calendar and Project Folders/Chat Threads utility rail.
- Bottom: Sandbox drop zone with large centered plus button.

## Styling rules

- Background: `#000000`.
- Panel/card fill: satin black `#050505` to `#080808`.
- Border: thin metallic silver `rgba(184,184,184,.45)`.
- Hover/focus: electric blue `#1677ff` glow.
- Active status: green.
- Inactive/off status: red.
